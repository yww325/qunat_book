# 第2章 金融计量经济学 I：线性回归（Econometrics I: Linear Regression）

## 1. 这一章解决什么问题？

- 用回归作为量化研究的“螺丝刀”：
  - 描述变量关系（相关/因果的候选结构）
  - 估计因子暴露、风险暴露
  - 评估信号有效性（解释力/显著性/稳定性）
- 同时强调：金融数据厚尾、异方差、共线性常见，**不做诊断=结果不可用**。

## 2. 核心工具与公式（最少但够用）

- 协方差/相关：
  - Cov(X,Y)=E[(X-μx)(Y-μy)]
  - ρ=Cov(X,Y)/(σxσy)
- 线性回归：Y=Xβ+ε
- OLS：β̂=(X'X)⁻¹X'Y

理解要点：回归本质是把Y投影到X张成的空间；共线性意味着“投影方向几乎重合”，估计会极不稳定。

## 3. 回归诊断（实战必做清单）

1) 残差是否像噪声？（均值≈0、无系统结构）
2) 异方差：波动是否随X/时间变化？
   - 用White / Breusch–Pagan等检验
   - 处理：稳健标准误、加权最小二乘、变换变量
3) 自相关（时间序列场景）：
   - Newey–West调整标准误
4) 多重共线性：
   - VIF过高→系数不稳
   - 处理：降维（PCA）、正则化、因子正交化/合并
5) 异常点/高杠杆点：
   - Winsorize / robust regression

## 4. 稳健回归（为什么金融里更该用）

- 金融收益/指标常见厚尾与极端值，OLS对极端点非常敏感。
- 可选方法：
  - Huber损失（折中L2与L1）
  - 分位数回归（关注中位数/尾部）
  - M-估计迭代

落地建议：你的研究报告里至少同时给出 OLS + 稳健回归（或稳健t值），否则容易“显著性幻觉”。

## 5. 树模型（CART）与非线性补充

- 树模型优点：能自动捕捉非线性与交互项。
- 风险：极易过拟合。
- 必做：交叉验证、剪枝、严格样本外。

## 6. 随机矩阵/高维直觉（与量化关联点）

- 在高维（因子多、股票多、样本短）环境下：
  - 协方差矩阵/回归系数噪声很大
  - 特征值谱可以用来识别“噪声区间”并做清洗（提高稳定性）

## 7. A股/港股落地提示

- 因子研究常用横截面回归：注意行业/市值等控制变量，否则你测到的可能是风格暴露。
- 财务因子：要处理披露滞后（否则前视偏差会让回归显著性虚高）。
- 港股流动性分化大：交易成本变量往往和很多因子相关，回归里最好加入流动性控制。
