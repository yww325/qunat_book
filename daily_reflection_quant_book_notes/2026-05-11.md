## 第一章：Review of Literature on Multifactor Asset Pricing Models

### 本章先导

本章由 Mario Pitsillis 撰写，回顾了多因子资产定价模型的文献发展脉络，是全书的理论基石。核心问题是：**为何 CAPM 这种单因子模型不够，而需要多因子模型？**答案在于系统性风险来源多元化——宏观经济增长、通胀、利率、信用利差等均能独立解释资产超额收益，而非仅仅是市场β。

### 逐节要点

#### 1.1 CAPM 的局限性与多因子起源

- **CAPM（资本资产定价模型）**：$E(R_i) = R_f + \beta_i \cdot (E(R_m) - R_f)$，只依赖市场投资组合的单一β。
- 问题：大量实证研究发现，存在"异象"（anomalies）——小市值股票、低估值股票、动量股持续跑赢市场，这些无法用单一市场β解释。
- Ross (1976) 提出的 **APT（套利定价理论）** 为多因子框架奠基：资产收益由多个因子共同驱动，定价关系在无套利条件下成立。

#### 1.2 无套利定价理论（APT）

- APT 的核心假设：因子暴露（factor loadings）$\beta_i$ 足够多时，不存在系统性套利机会。
- 收益生成过程：$R_i = \alpha_i + \sum_j \beta_{ij} F_j + \varepsilon_i$
- 若 $\alpha_i = 0$（对所有 i），则因子风险溢价由协方差矩阵隐式决定。
- APT vs. CAPM：APT 更灵活，不要求存在"真"市场组合；因子可以是任何系统性风险来源。

#### 1.3 因子选择的三种范式

| 范式 | 代表方法 | 说明 |
|------|------|------|
| **宏观因子模型** | Fama-French (1993) 三因子 | 因子来自宏观经济变量（MKT、SMB、HML） |
| **基本面因子模型** |BARRA、Citigroup 模型 | 因子来自公司财务特征（规模、估值、动量等） |
| **统计因子模型** | PCA（主成分分析） | 因子从历史收益协方差矩阵提取，不具经济解释性 |

#### 1.4 因子模型估计方法

- **时间序列回归**（GRS 检验，Fama-MacBeth 截面回归）
- **横截面回归**：先做时间序列得到 $\beta_{ij}$，再做截面回归估计因子风险溢价 $\lambda_j$
- 关键问题：**因子数量 vs. 资产数量**：当因子数接近资产数时，$\beta$ 矩阵病态，OLS 估计失真。

#### 1.5 文献中的典型因子

- **市场因子（MKT）**：超额市场收益
- **规模因子（SMB）**：Small Minus Big，小盘股溢价
- **价值因子（HML）**：High Minus Low，账面市值比溢价
- **动量因子（MOM）**：过去 12 个月收益动量
- **流动性因子**：Amihud (2002) 非流动性比率
- **盈利因子**：ROE、盈利率

#### 1.6 因子正交化问题

- 多因子模型中，因子间往往存在高度相关性（如 SMB 与规模高度相关）。
- **正交化处理**：对因子进行 Gram-Schmidt 正交化，剥离冗余信息，使每个因子独立贡献解释力。
- 但正交化可能破坏因子的经济含义，需谨慎使用。

### 章末总结

本章系统梳理了多因子资产定价模型的理论根基。CAPM 的单一β无法捕捉市场异象，APT 提供了多因子框架的理论合法性。文献中存在宏观因子、基本面因子、统计因子三类选择路径。估计方法分为两步：时间序列回归求β，再截面回归求风险溢价。关键挑战在于因子选择的经济解释性与统计有效性之间的权衡。

### 可背诵版

> CAPM 只用一个市场β定价，无法解释小市值、低估值、动量等异象。APT 通过假设多因子收益生成过程 $R_i = \alpha_i + \sum_j \beta_{ij}F_j + \varepsilon_i$，允许无套利条件下多因子定价。因子选择有宏观（FF三因子）、基本面（BARRA）、统计（PCA）三条路。估计用两步法：时序回归得β，截面回归得溢价。

### 术语卡片

| 术语 | 英文 | 定义 |
|------|------|------|
| 资本资产定价模型 | CAPM | $E(R_i) = R_f + \beta_i(E(R_m)-R_f)$ |
| 套利定价理论 | APT | 无套利条件下多因子定价框架 |
| 因子载荷 | Factor loading (β) | 资产对某因子的敏感度 |
| 因子风险溢价 | Factor risk premium (λ) | 投资者承担因子风险的额外补偿 |
| Fama-French 三因子 | FF3 | MKT + SMB + HML |
| GRS 检验 | Gibbons-Ross-Shanken test | 联合假设检验（alpha=0 且定价因子有效） |
| 异象 | Anomaly | 无法被现有定价模型解释的收益规律 |

## 第二章：Estimating UK Factor Models Using the Multivariate Skew Normal Distribution

### 本章先导

本章由 C.J. Adcock 撰写，核心贡献是**用多元偏态正态分布（MSN）替代传统正态分布来估计英国因子模型**。传统因子回归假设残差服从正态分布，但股市收益往往呈现尖峰厚尾（kurtosis）和偏度（skewness）——这意味着极端事件比正态假设下更频繁。MSN 分布可以同时捕捉这两个特征，从而得到更准确的因子风险溢价估计量。

### 逐节要点

#### 2.1 为什么要超越正态分布

- 金融收益的**四大特征**：
  1. **尖峰（Leptokurtosis）**：峰度 > 3，尾部比正态分布更厚
  2. **偏度（Skewness）**：收益分布不对称，负偏更常见（"跌多涨少"）
  3. **波动率聚类（Volatility clustering）**：大波动往往跟随大波动
  4. **均值回复（Mean reversion）**：长期来看收益趋于均值
- 若忽略偏度和峰度，参数估计会产生系统性偏差，尤其在尾部风险度量（VaR、CVaR）中造成严重低估。

#### 2.2 多元偏态正态分布（MSN）定义

- **标准正态**：$Z \sim N(0, \Sigma)$
- **MSN 分布**：$Y = \xi + \Omega^{1/2}Z + \tau \cdot |Z|$，通过加入 $|Z|$ 项引入偏度。
- 参数：$\xi$（位置向量），$\Omega$（尺度矩阵），$\tau$（偏度向量）
- 当 $\tau = 0$ 时，MSN 退化为标准正态分布——正态是 MSN 的特例。

#### 2.3 UK 因子模型估计结果

- 文章使用英国股票收益数据（FTSE 成分股），估计三因子模型。
- **关键发现**：使用 MSN 分布假设后，因子载荷的估计精度显著提升；风险溢价的置信区间收窄。
- 部分因子（如 SMB）在正态假设下不显著，但在 MSN 假设下显著——说明正态假设掩盖了真实关系。

#### 2.4 估计方法：最大似然估计（MLE）

- 目标函数：最大化 MSN 分布下的对数似然函数
- 数值优化：由于 MSN 密度函数复杂，需使用迭代优化（如 BFGS 算法）
- 初始值设定：先用正态估计量作为 MSN 估计的初始值，保证收敛。

#### 2.5 偏度校正的实践意义

- 对**对冲基金和期权组合**，尾部风险至关重要。
- MSN 模型能更准确估计 **条件 VaR（CVaR）**，为风险预算提供更可靠的依据。
- 在 **组合优化** 中，使用 MSN 可避免低估极端损失导致的过度杠杆。

### 章末总结

本章挑战了因子回归中"残差正态"的标准假设。英国股市数据表明，采用 MSN 分布能更准确捕捉收益分布的偏度和峰度，从而改善因子载荷估计精度，部分因子在正态假设下被误判为不显著。这是将 distributional assumption 融入量化模型的典型案例。

### 可背诵版

> 正态分布假设掩盖了金融收益的偏度和峰度。MSN 分布 $Y = \xi + \Omega^{1/2}Z + \tau|Z|$ 将正态作为特例，通过偏度参数 $\tau$ 引入不对称性。在英国三因子模型中，MSN 假设使因子估计更精确，部分原本不显著的因子变得显著，风险溢价置信区间收窄。

### 术语卡片

| 术语 | 英文 | 定义 |
|------|------|------|
| 多元偏态正态分布 | MSN (Multivariate Skew Normal) | 加入绝对值项引入偏度的多元正态扩展 |
| 尖峰厚尾 | Leptokurtosis | 峰度 > 3，尾部概率高于正态分布 |
| 偏度 | Skewness | 分布对称性度量，负偏 = 左尾更长 |
| 条件风险值 | CVaR (Conditional VaR) | 极端损失的平均值（VaR 条件期望） |
| 最大似然估计 | MLE | 通过最大化似然函数估计参数 |
| BFGS 算法 | Broyden–Fletcher–Goldfarb–Shanno | 拟牛顿法，用于非线性优化 |

## 第三章：Misspecification in the Linear Pricing Model

### 本章先导

本章由 Ka-Man Lo 撰写，聚焦**线性定价模型中的误设定（Misspecification）问题**。当研究者使用错误的因子集、或因子载荷不满足正确模型的结构时，定价模型会给出误导性结论。本章系统分析了误设定的来源、检验方法及其对组合配置的影响，强调"模型选择错误"与"参数估计误差"是两类不同的问题。

### 逐节要点

#### 3.1 误设定的定义与来源

- **误设定（Misspecification）**：模型假设与真实数据生成过程（DGP）不一致。
- 来源一：**遗漏变量（Omitted variables）**：遗漏了真实存在的系统性因子 → 因子风险溢价估计有偏。
- 来源二：**冗余变量（Redundant variables）**：纳入的因子可被其他因子线性表示 → 因子共线性，标准误膨胀。
- 来源三：**时变因子载荷（Time-varying betas）**：假设 $\beta_i$ 恒定，但实际随市场状态变化 → 跨期预测失效。

#### 3.2 误设定的检验

- **规格化检验（Specification tests）**：J-test、Encompassing test
- **嵌套检验（Nested test）**：比较约束模型与非约束模型（如 CAPM vs. FF3）
- **Overidentifying restrictions**：当工具变量数 > 内生变量数时，检验工具有效性（Sargan-Hansen J-test）
- 核心思想：若模型正确，所有因子的超额收益应能由因子暴露解释；残差应与因子正交。

#### 3.3 对风险溢价估计的影响

- 遗漏关键因子时，截面回归得到的 $\lambda_j$ **系统性地遗漏了被遗漏因子的溢价**，导致定价误差。
- 案例：若真实模型需要加入流动性因子，但研究者只用了 FF3，则 FF3 的三个因子溢价均会被"污染"，不再反映真实风险补偿。

#### 3.4 稳健推断策略

- **因子扩充检验（Factor augmentation）**：逐步加入新因子，观察 $\lambda$ 是否稳定
- **交叉验证**：将样本分为"训练集"和"测试集"，检验模型外样本预测能力
- **贝叶斯模型平均（BMA）**：不选单一最优模型，而是对所有可能模型加权平均，降低误选风险

#### 3.5 时变因子载荷的建模

- **条件 CAPM**：$\beta_i(t) = \beta_{i,0} + \gamma_i' z_{t-1}$，其中 $z_{t-1}$ 为滞后信息变量
- **状态转移模型**：因子载荷在"高波动/低波动"两状态间切换
- **GARCH-in-mean 模型**：波动率的时变预期影响收益的条件均值

### 章末总结

误设定是多因子模型应用中的核心风险。遗漏关键因子导致系统性定价偏差，纳入冗余因子浪费自由度并膨胀标准误。检验方法包括 J-test、Overidentifying restrictions 等。稳健策略包括因子扩充检验、交叉验证和贝叶斯模型平均。时变因子载荷需要条件因子模型来捕捉动态风险结构。

### 可背诵版

> 线性定价模型误设定有三：遗漏因子（溢价有偏）、冗余因子（标准误膨胀）、时变β（跨期失效）。检验用 J-test 和 Overidentifying restrictions。稳健推断：因子扩充检验、交叉验证、贝叶斯模型平均。时变β用条件因子 $E[R_i|t]=a_i+b_i'F_t+\gamma_i'z_{t-1}$ 建模。

### 术语卡片

| 术语 | 英文 | 定义 |
|------|------|------|
| 误设定 | Misspecification | 模型假设与真实数据生成过程不一致 |
| 遗漏变量偏误 | Omitted variable bias | 遗漏相关变量导致的参数估计偏误 |
| J 检验 | J-test | 检验嵌套模型中约束是否有效 |
| Overidentifying restrictions | 过度识别约束 | 工具变量数 > 内生变量数时的有效性检验 |
| 贝叶斯模型平均 | BMA (Bayesian Model Averaging) | 对多模型加权平均而非选择单一最优模型 |
| 条件因子模型 | Conditional factor model | β 随时间/状态信息而变的因子模型 |

## 📝 反思

### 量化策略思考

- **因子模型的选择不是中性的**：选择哪些因子决定了组合的风险预算和预期收益。当前主流 FF3+ 模式（加入动量、盈利、质量等因子）并非"自然真理"，而是历史演进和学术共识的产物。Quant 研究者应持续审视因子集合是否有经济逻辑支撑，还是仅仅是数据挖掘的结果。
- **分布假设比想象中重要**：第二章的 MSN 模型结果表明，正态假设可能系统性地低估尾部风险。当策略涉及期权卖方或杠杆组合时，尾部风险的精确度量至关重要——误判可能导致爆仓。

### 与已有知识体系的关联

- 因子正交化（第一章）与机器学习中的特征工程高度相关：去除共线性本质上是在减少过拟合风险。
- 贝叶斯估计（第二章相关）也是当前量化前沿的热门方向——将主观先验（如 Litterman Minnesota Prior）与数据似然结合，比纯频率学派更稳定。

### 可改进的实践

### 今日代码目标：FF3 因子模型实现

```python
import pandas as pd
import numpy as np
import statsmodels.api as sm

# Fama-French 三因子回归
def ff3_regression(ret, factor_df):
    """
    ret: 资产超额收益 (T x N)
    factor_df: 因子数据 (MKT, SMB, HML) (T x 3)
    """
    X = sm.add_constant(factor_df)
    results = {}
    for asset in ret.columns:
        y = ret[asset]
        model = sm.OLS(y, X).fit()
        results[asset] = {
            'alpha': model.params['const'],
            'beta_mkt': model.params['MKT'],
            'beta_smb': model.params['SMB'],
            'beta_hml': model.params['HML'],
            'r_squared': model.rsquared,
            'pvalues': model.pvalues.to_dict()
        }
    return pd.DataFrame(results).T

# 截面回归：估计因子风险溢价
def cross_section_premium(betas, exp_rets):
    """
    betas: N x K 因子载荷矩阵
    exp_rets: N 资产预期收益
    """
    lambda_k = np.linalg.lstsq(betas, exp_rets, rcond=None)[0]
    return lambda_k

# 因子正交化（Gram-Schmidt）
from numpy.linalg import qr

def gram_schmidt_orthogonalize(factors):
    """对因子列进行正交化"""
    Q, R = qr(factors.values)
    return pd.DataFrame(Q, columns=factors.columns, index=factors.index)
```
