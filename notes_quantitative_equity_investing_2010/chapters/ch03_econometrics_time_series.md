# 第3章 金融计量经济学 II：时间序列（Econometrics II: Time Series）

## 1. 这一章解决什么问题？

- 量化投资不仅要做横截面（选股），还要处理时间维度：
  - 因子收益是否随时间变化？
  - 波动/相关性是否时变？
  - 变量之间是否存在长期均衡（协整）？
- 本章提供一套时间序列建模工具箱，并强调“平稳性”是建模前置条件。

## 2. 平稳性（建模前的闸门）

- 弱平稳：均值、方差、协方差不随时间变。
- 常用检验：
  - ADF（原假设：存在单位根，非平稳）
  - PP
  - KPSS（原假设：平稳）

实战建议：ADF与KPSS搭配用，避免单检验误判。

## 3. AR/MA/ARMA（单变量基础）

- AR(p)：Y_t = φ₀ + Σ φᵢ Y_{t-i} + ε_t
- MA(q)：Y_t = μ + ε_t + Σ θᵢ ε_{t-i}
- ARMA(p,q)：二者结合

用途：
- 用于建模均值过程（例如因子收益序列）
- 更常见的是扩展到波动过程（虽本章重点不在GARCH，但思路相关）

## 4. VAR / Granger / IRF（多变量动态）

- VAR：多变量互相滞后解释，用于研究变量联动（如利率、信用、行业收益）。
- Granger因果：检验“过去的X是否帮助预测Y”。
- IRF脉冲响应：看一个冲击对系统的动态影响路径。

注意：Granger不是哲学意义因果，只是预测意义因果。

## 5. 协整与VEC（长期关系）

- 两个（或多个）非平稳序列，其线性组合可能平稳 → 存在长期均衡。
- 常用：Engle-Granger两步、Johansen。
- VEC：用“偏离长期均衡”驱动短期修正。

落地：价差、相对估值、行业间长期关系等研究会用到。

## 6. 滞后阶数选择

- AIC/BIC/HQ 等信息准则。
- 实战原则：宁可稍短+稳健性检验，也别无限加长导致过拟合。

## 7. 非线性与区制（为什么策略会失效）

- 门限/平滑转移/Markov区制模型：描述“市场在不同状态下行为不同”。
- 量化策略常见现象：
  - 牛市动量更强、熊市价值/质量更占优
  - 波动状态切换导致风险模型失真

## 8. A股/港股落地提示

- 波动与相关性强时变：
  - 做组合优化时建议用滚动估计 + 收缩/稳健化，避免某段行情把协方差估计带偏。
- 协整/配对策略在港股ADR/双重上市、A/H溢价研究中常见，但交易可行性（借券、成本）要先评估。
