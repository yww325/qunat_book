# F03_线性因子模型的估计误差与稳健性（PDF 81–130 页）

## 0. 本卷覆盖范围与定位
本卷（F03）对应《Linear factor models in finance (2005)》PDF **81–130 页**附近内容。主题集中在：
- 线性因子模型的计量估计（截面 vs 时序）
- 估计误差（尤其是误差在变量 EIV / β 估计噪声）
- 收缩/正则化的直觉与实践（Ridge/Lasso/Elastic Net/JS）
- 因子可解释性与“统计因子”（PCA/旋转）的取舍
- 因子暴露稳定性与结构突变（rolling / break tests）
- 向高频/短周期（如 15m）迁移时要特别小心的噪声问题

> 读者提示：Knight & Satchell 的核心关切之一是**“因子模型并不难写，难的是估计误差和样本外稳健性”**。这对我们港股 15m 扫描/回测项目非常关键：短周期下样本更噪、结构更易漂移，因子/阈值如果不做稳健处理，很容易在 walk-forward 外崩。

---

## 1. 章节结构/主题概览（本段的逻辑主线）

1) **模型设定**：用线性结构刻画收益分解（公共因子 + 特异项），强调“可交易因子/可解释因子”和“统计因子”的差异。

2) **两类回归视角**：
- **时序回归（TSR）**：对每个资产用时间序列估计 β
- **截面回归（CSR）**：对每个时间点用横截面估计因子收益（或风险溢价）
并讨论两者在某些条件下的联系与误差传导。

3) **估计误差/EIV**：β 是估计出来的而不是已知的，因子也可能是代理/模拟出来的；于是会出现：
- 衰减偏误（attenuation bias）
- 多重共线性导致的不稳定估计
- 小样本下高方差

4) **收缩与正则化**：用“带偏换稳”的方式，把 β、因子风险溢价或协方差的估计变得更稳健。

5) **稳健性检验**：rolling、断点检验、条件数监控等。

## 2. 关键概念/定义/公式（抓住可落地的那部分）

### 2.1 线性因子模型（基本形式）
常见形式（以超额收益为例）：

- **时序表示（给定因子收益）**
  \[
  R_{it} = \alpha_i + \beta_i^\top f_t + \varepsilon_{it}
  \]
  其中：
  - \(f_t\)：K 维因子收益（或因子冲击）
  - \(\beta_i\)：资产 i 对因子暴露
  - \(\varepsilon_{it}\)：特异项

- **截面表示（给定暴露，估计风险溢价/因子收益）**
  在某一时点 t：
  \[
  R_t = \alpha + B\,\lambda_t + e_t
  \]
  - \(R_t\)：N 维横截面收益
  - \(B\)：N×K 暴露矩阵
  - \(\lambda_t\)：K 维“当期因子回报/风险溢价”（不同文献符号不同）

> 实务上：TSR 更像“估 β”，CSR 更像“估当期因子收益/风险溢价”。


### 2.2 两阶段法（Fama–MacBeth 思想的核心风险）
典型流程：
1) 第一步 TSR：\(\hat\beta_i\) 来自历史窗口回归
2) 第二步 CSR：用 \(\hat\beta\) 做横截面回归得 \(\hat\lambda\)

风险点：\(\hat\beta\) 是带噪声的 → 第二阶段会产生 **EIV（误差在变量）** 问题。


### 2.3 误差在变量（EIV）与“衰减偏误”直觉
若回归解释变量（这里是 β 或因子代理）含测量误差：
\[
\hat x = x + u
\]
在经典条件下会导致系数估计向 0 收缩（衰减偏误），同时标准误也会被误判。

> 对我们：如果把某些“因子/指标”当作解释变量（哪怕只是 score 里的组件权重），这些组件自身噪声很大时，会让阈值/权重在样本外失效。


### 2.4 收缩/正则化（用偏差换方差）
- **Ridge（L2）**：
  \[
  \hat\theta=\arg\min_\theta\|y-X\theta\|_2^2 + \lambda\|\theta\|_2^2
  \]
  适用于：共线、K 不小、希望稳定系数。

- **Lasso（L1）**：
  \[
  \hat\theta=\arg\min_\theta\|y-X\theta\|_2^2 + \lambda\|\theta\|_1
  \]
  适用于：希望稀疏（自动做变量选择）。

- **Elastic Net（L1+L2）**：
  适用于：高度共线 + 想稀疏。

- **James–Stein / Bayes shrinkage（思想层面）**：
  多参数同时估计时，向共同均值收缩可整体降低 MSE。


### 2.5 因子正交化/PCA 与可解释性权衡
- PCA 因子能提高拟合/降噪，但会牺牲经济解释性。
- 旋转（如 Varimax）尝试在“解释性”和“统计正交性”之间折中。

> 对我们：更重要的是稳定性（样本外），其次才是解释性；但解释性可以帮助做风险控制与监控（出现 regime change 时更快定位）。

## 3. 作者论证链条（我按“为什么这么做→假设→局限”复盘）

### 3.1 为什么要同时讲 TSR 与 CSR？
- **原因**：因子模型要落地必须可估计；不同估计路径对误差的敏感性不同。
- **隐含假设**：
  - 因子结构相对稳定
  - 误差项满足一定独立性/同方差或可被修正
- **局限**：现实市场存在：
  - 异方差、厚尾、跳跃
  - β 时变
  - 因子代理不完美
  这些会把“理论等价”变成“实践不等价”。


### 3.2 为什么作者强调估计误差/EIV？
- **原因**：因子模型的“漂亮解释”往往来自样本内；一到样本外就崩，多半是估计误差与结构漂移。
- **关键点**：
  - β 估计误差会向后传导到因子溢价/预测
  - 因子共线会让系数高度不稳定
- **局限**：高频（15m）下样本噪声更大，EIV 更严重；若不做收缩/稳健标准误，结论会“看起来显著但不可交易”。


### 3.3 为什么引入收缩/正则化？
- **原因**：在有限样本、多因子、共线条件下，OLS 的方差会爆炸；收缩能让样本外更稳。
- **假设**：
  - 真实参数在某种意义上“接近 0/接近均值/稀疏”
  - 或者我们宁愿接受偏差换稳定
- **局限**：收缩强度 λ 选不好会欠拟合；并且对 regime change 仍无解（只能缓和）。


### 3.4 为什么要做暴露稳定性检验？
- **原因**：β 时变时，“长期估计的 β”会误导短期。
- **方法**：rolling、断点检验、对条件数/共线的监控。
- **局限**：检测到断点不等于知道新 regime 的正确模型；只能触发“重新估计/降杠杆/收紧阈值”。

## 4. 小节落地建议（对港股 15m 项目怎么改）

> 我按“估计误差/稳健性”这条主线，给出可直接进代码/参数的建议。

### 4.1 把 score / 触发逻辑当成“简化的因子模型”来做稳健化
我们现在的 scan 触发，本质上是：
- 用若干信号（DD、KDJ J、RSI 等）去解释/预测“后续反弹概率”
这在统计上就是一个小型因子/特征模型。

可落地改进：
1) **加入收缩思想**：
   - 当前 `score_params` 固定权重（w_dd, w_j, w_rsi_os）。建议引入一个“收缩到默认权重”的机制：
     - 当最近 20 天样本很少（触发少）→ 权重更接近默认
     - 当样本充足、稳定 → 允许权重偏离
   - 这相当于对参数做 ridge/JS 的先验收缩。

2) **把“阈值/权重”与样本外稳定性挂钩**：
   - 在 walk-forward 输出里，额外记录每个参数组的“稳定性指标”：
     - test strong/stable 的方差
     - rolling 命中率的波动
     - 触发间隔（避免集中在少数日）


### 4.2 用 rolling 的“暴露稳定性”思想检查策略是否 drift
对每个 ticker：
- 每 20 个交易日滚动一次（或每周一次）输出：
  - `dd_TierB_triggers`、stable_rate、strong_rate 的 rolling 序列
  - 触发数不足时标记为“不可评估”

落地做法：
- 在 backtest 模式增加一个可选参数（未来可加到 JSON）：
  - `--rolling-report 20d` 或 `rolling_window_days=20`
  - 输出 rolling 的稳定性摘要（不必画图，先输出几列即可）

风险控制用途：
- 当 rolling strong_rate 急剧下降 → 自动提高 `score_min` 或暂停该 ticker。


### 4.3 解决“无触发静默”的同时，保证触发信息可操作
我们已实现：无触发不发 Telegram。
进一步建议：
- **触发时消息中增加两个字段**（便于你快速判断）：
  1) “近 60d strong_rate / stable_rate”（来自 backtest 结果缓存或上次回测）
  2) “当前触发属于 W=5 还是 W=8 或同时触发”（已在 triggers 字段里，但可以提取成短标签）

这样你看到消息就知道：
- 这是哪个统计口径下“胜率较高”的信号，是否值得看盘。


### 4.4 EIV 视角：减少高频噪声对信号的误判
15m 下最常见的问题是“最后一根 bar 噪声触发”。可做：
- **确认 bar 已收盘**（yfinance 的时间戳可能是 UTC，注意对齐港股交易时段）
- 对 DD/RSI/J 使用“短窗口平滑”：
  - 例如对 Close 做 2-bar median filter
  - 或对 RSI 使用 3-bar EMA

这些是“降噪”，相当于用 PCA/预平均的思想减少微观结构噪声。

## 5. TL;DR（≤10 行）
1) 本段强调：因子模型最难的是 **估计误差** 与 **样本外稳健性**。
2) TSR/CSR 两种估计视角在理想条件下有关联，但实践中误差传导不同。
3) **EIV 会导致衰减偏误**，高频下更严重。
4) **收缩/正则化（Ridge/Lasso/JS）** 的核心是“带偏换稳”。
5) PCA/正交化能降噪，但要权衡解释性。
6) 必做 rolling / 断点检验来监控 β/信号稳定性。
7) 对港股 15m：建议把 score/阈值当作参数做“收缩+稳定性约束”，并增加 rolling 稳定性报告。
